This is the compiled version of the project.

Pre-requisites:
- Outlook 2010 (2013 may also work)
- Windows Installer 4.5+
- .NET Framework 4.0+
- Visual Studio Run-times for Office 2010/2013


Automatically install:
Copy the files to C:\TEMP\Publish
Share C:\TEMP\Publish as "PUBLISH"
Run SETUP.EXE
(This should install the pre-reqs if necessary)


Manually install:
Copy the files to C:\TEMP\Publish
Share C:\TEMP\Publish as "PUBLISH"
Double-click the VSTO file to install
